# 🧠 PsyCare

Sistema de gestão para clínicas de psicologia, focado em organização de atendimentos, controle de pacientes e sessões online.

---

## 🚧 Status

🚧 Projeto em desenvolvimento

---

## 💡 Sobre o projeto

O PsyCare foi desenvolvido com o objetivo de auxiliar profissionais da área da psicologia na gestão de seus atendimentos, oferecendo uma solução prática e organizada para o dia a dia clínico.

---

## ✅ Funcionalidades implementadas

- 👤 Cadastro de pacientes
- 📅 Agenda de sessões
- 📄 Prontuário clínico
- 💻 Estrutura para sessões online

---

## 🚀 Em desenvolvimento

- 📊 Dashboard com métricas
- 🔐 Sistema de autenticação completo
- 🎨 Melhorias de UI/UX
- 📈 Relatórios e análises

---

## ⚙️ Tecnologias

- JavaScript
- Node.js
- API REST
- HTML & CSS

---

## 🎯 Objetivo

Criar uma plataforma completa para gestão de clínicas de psicologia, com foco em usabilidade, desempenho e experiência do usuário.
